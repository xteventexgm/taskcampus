# TaskCampus

## Descripción del proyecto
[cite_start]TaskCampus es una aplicación web diseñada para que los estudiantes universitarios puedan organizar sus actividades académicas, registrar fechas de entrega y controlar su estado de avance. El proyecto fue planificado bajo el enfoque Spec Driven Development y construido con TypeScript y Tailwind CSS en el frontend.

## Requisitos previos
Para poder ejecutar y modificar este proyecto en tu computadora, es necesario contar con:
1. **Node.js** instalado en el sistema.
2. **TypeScript** instalado de forma global (se puede instalar ejecutando `npm install -g typescript` en la terminal).
3. Un editor de código como Visual Studio Code con la extensión **Live Server**.
