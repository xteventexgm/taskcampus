# TaskCampus

## Descripción del proyecto
TaskCampus es una aplicación web diseñada para que los estudiantes universitarios puedan organizar sus actividades académicas, registrar fechas de entrega y controlar su estado de avance. El proyecto fue planificado bajo el enfoque Spec Driven Development y construido con TypeScript y Tailwind CSS en el frontend.
